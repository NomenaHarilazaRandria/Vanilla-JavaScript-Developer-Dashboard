# 📊 Developer Dashboard

A responsive developer dashboard built with **HTML, CSS and Vanilla JavaScript**.

The application allows you to manage tasks through a modal Kanban board while displaying real-time statistics, progress bars, recent completed tasks, current tasks, a live digital clock and a simulated weather widget.

This project was created to practice JavaScript DOM manipulation, application architecture and state management before learning React.

---

## ✨ Features

- 📋 Task management
- ➕ Add new tasks
- 🚚 Move tasks through three stages:
  - To Do
  - Doing
  - Done
- 🗑 Delete completed tasks
- 💾 Automatic saving with LocalStorage
- 📈 Live dashboard statistics
- 📊 Progress bars
- ✅ Last completed tasks
- 🚧 Current tasks in progress
- 🌤 Simulated weather widget (Async/Await)
- 🕒 Live digital clock
- 📱 Responsive design
